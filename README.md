# City Hospital – Data Preprocessing & Exploratory Analysis

**Final Project | Data Preprocessing and Visualization | NOVA IMS**

This project was developed as the **Final Project for the Data Preprocessing and Visualization course** in the **Bachelor’s in Data Science at NOVA IMS**.

## Project Objective

The goal of this project is to prepare hospital patient data for future analytical tasks. The work focuses on transforming raw operational data into an **Analytical Base Table (ABT)** that can later be used for **clustering and predictive modeling**.

## Project Tasks

The project is divided into three main parts:

### 1. Data Understanding & Cleaning

* Exploration of dataset structure
* Handling missing values
* Fixing inconsistent data types
* Detecting and treating outliers

### 2. Feature Engineering & ABT Construction

Creation of an **Analytical Base Table (ABT)** with **one row per patient** including:

* Demographic attributes (age, gender, education, profession)
* Aggregated visit behaviour
* Satisfaction indicators
* Recency and activity metrics

### 3. Exploratory Data Analysis

Visualizations were created to understand:

* Patient demographic distributions
* Department usage patterns
* Consultation activity over time
* Satisfaction levels across departments
* Patient segmentation patterns
